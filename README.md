# 📎 Link Shortener

Shortens links.

## 📦 Usage

Clone the [repository](https://github.com/1nter-p/link-shortener), install
dependencies and run.

```sh
git clone https://github.com/1nter-p/link-shortener
cd link-shortener
pip3 install -r requirements.txt
python3 -m app
```
